#Exploratory Data Analysis - Paisabazaar (Capstone Project)
#Credit Score Classification & Risk Analysis – Paisabazaar Case Study

## Project Overview
This project performs structured Exploratory Data Analysis (EDA) 
on customer financial data to analyze and understand credit score classification in the context of **Paisabazaar**, a financial services platform.


## Objective
To identify financial and behavioral factors influencing credit score categories 
and derive business insights to improve credit risk assessment and product recommendations for Paisabazaar.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Google Colab

## Key Insights
- Debt and delayed payments strongly influence credit risk.
- Longer credit history reduces financial risk.
- Interest rates follow a risk-based pricing pattern.
- Credit utilization acts as an independent risk indicator.
